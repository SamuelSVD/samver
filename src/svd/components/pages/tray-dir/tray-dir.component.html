How To Use

When you first start up TrayDir, you will be greeted with the form containing one instance, New Instance. Each instance will have its own icon in the system tray. Each instance can be configured to have multiple items inside which work as shortcuts to different file or folder locations... That was a mouthful

However you decite to configure the items in the main window is how the tray menu will look when you right click, meaning you can have custom shortcuts in whatever structure you want! This might be easier to understand with an example

Adding New Items

You can add new items by making use of the top buttons. You can add a new file link, new folder link, or new virtual folder to the tree of items.

Creating New Instances

You can create a new instance by clicking on the + tab, or by clicking on Instance | New from the menu bar
