TrayDir

TrayDir is a neat little program intended for Windows use.

Its main use is for programming situations where you would like quick access to a script to run. Something as simple as 'run task' or 'change ip'. This little tool will allow you to have quick access to anything you would like.

Download

TrayDir v1.0.5.3

Example

Let's say I have a folder that I use for running all sorts of scripts. Some related to changing my IP, some related to web development, and another that I use for fun.

Figure 1: Folder Structure

I want easy access to this folder to be able to run different programs. This sort of thing happens in everyday dev work.

All you have to do is run the program, point it to the folder you would like access to, and TADA! The folder will be easily accessible from your system tray.
